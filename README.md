
# EFR'ATP


A web appplication aiming to display on school's TVs the next departures of public transports nearby Efrei.

- Metro Line 7
- Tramway Line 7
- Bus 172
- Bus 380

## Features

- Next departures
- Live traffic information



## API Reference


| API       | Description                       |
| :-------- | :-------------------------------- |
| `IDFM`      | Next departures |
| `SNCF - Transilien`      | Traffic information |
